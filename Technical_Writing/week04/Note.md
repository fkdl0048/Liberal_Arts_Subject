# 4주차

## 복습

동사를 선택하기 위해선 다음의 순서를 기억하자

- 동사 확인
  - 동작동사
    - 일반
    - 진행
  - 상태 동사
    - 일반
- 타임 워드 확인
- 현재 시제
  - 현재 완료 (have pp)
    - 결과 how many, much
  - 현재 완료 진행 (have been ing)
    - 진행 how long

### 과제 복습

## 과제 완료/ 과거 완료 진행형

- past perfect: 주어 + had + pp
- past perfect continuous: 주어 + had been + ing
